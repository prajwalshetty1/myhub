# Icons

This folder should contain PWA icons in the following sizes:
- 72x72
- 96x96
- 128x128
- 144x144
- 152x152
- 192x192
- 384x384
- 512x512

For now, the app will work without icons, but PWA installation features may be limited.

To generate icons:
1. Create a 512x512 master icon (PNG format)
2. Generate all required sizes using an icon generator tool
3. Place them in this folder with names: icon-{size}.png
