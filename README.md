## üî¨ How It Works

### Compatibility Calculation
- **Zodiac Compatibility**: Based on astrological element compatibility (Fire, Earth, Air, Water)
- **Numerology**: Calculates life path numbers from birth dates
- **Combined Score**: Weighted average of zodiac and numerology compatibility

### Biorhythm Calculation
Biorhythms are biological cycles that affect human behavior:
- **Physical Cycle**: 23 days - affects strength, stamina, and coordination
- **Emotional Cycle**: 28 days - affects mood, creativity, and emotions
- **Intellectual Cycle**: 33 days - affects memory, alertness, and analytical thinking

### Conversation Topics
Topics are generated based on:
- Overall compatibility level
- Current biorhythm status for both partners
- Universal relationship building themes
